## Meeting formats

### Daily standup
```
### Ieri
- [cosa è stato fatto]

### Oggi
- [cosa si farà]

### Bloccanti
- [eventuali blocchi]
```

### Sprint planning
```
### Obiettivo dello sprint
[Una frase chiara]

### Task
- [task] → [assegnatario] → [story points]

### Capacity
[Quanto possiamo fare realisticamente]
```

### Retrospective
```
### Cosa ha funzionato
[3-5 punti]

### Cosa migliorare
[2-3 punti, con soluzione proposta]

### Action items
[Cosa cambiamo per il prossimo sprint]
```
